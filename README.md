# Contacts-in-C
 جهات الاتصال هو مشروع صغير يقوم بحفظ جهات الاتصال في ملف معين من (اسم اول/اسم الاخير/رقم الهاتف/البريد الالكتروني) حيث يحتوي على:
- اضافة جهة اتصال
- البحث على جهة اتصال 
- تعديل جهة اتصال
- عرض جميع جهات الاتصال
- حذف جهة اتصال معينة
- حذف جميع جهات الاتصال
